# 🛡️ Cyber-Defense-System (Phase 1)

A low-level process monitoring engine developed in **C**. This project focuses on the efficient implementation of **Linked Lists** to manage system entities (processes) in real-time, simulating a cybersecurity environment.

## 🛠️ Tech Stack
- **Language**: C (C99 Standard)
- **Editor**: Neovim (LazyVim setup)
- **Environment**: WSL2 (Ubuntu)
- **Compiler**: GCC

## 📂 Project Structure
- `main.c`: The system's control center (Entry point).
- `process.c`: The core engine logic (Allocation, Insertion, Traversal).
- `process.h`: The "Contract" defining structures and function signatures.

🎓 Academic Context
This project was developed during my 3rd semester of Computer Engineering at IPRJ/UERJ. It serves as a practical application of Data Structures and Dynamic Memory Allocation concepts.
